Design Project 3 (Data Driven World)

The aim of this project was to use machine learning models to solve COVID-19 related problems

Task 1

Aim: Predict the number of covid related deaths
For this project, the death on the next day is predicted using several variables, and a linear regression method was used to form the model. The data from this dataset is sourced from the Our World in Data Organization and holds daily covid data from over 200 countries. This link can be found below:

Data

The countries used in our dataset was selected through k-nearest-neighbours to find closely related countries and the countries are visualised below


These countries data was then selected and further processed to train the model.

Task 2

Aim: Distinguish Fake News from Real News
For this project, a logistic regression model was used to determine if a news article contains fake or real news. The data from this dataset contains news articles from real and fake sources and contain some other information about each article:

Data

Some of the features used in this model is the length of the title, abstract and how many keywords or 'metawords' are identified in the post
